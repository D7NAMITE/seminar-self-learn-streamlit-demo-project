# Streamlit Data Visualization Demo

This project is the demo component of the New Tech Trend Self-Learn Assignment of the Seminar course (01219497) at Kasetsart University.

The dataset used in this project is available at Kaggle: [Superstore Dataset](https://www.kaggle.com/datasets/vivek468/superstore-dataset-final?resource=download).

For a deeper technical details of Streamlit and a "Cheat Sheet" of commands, refer to the [Official Streamlit Documentation](https://docs.streamlit.io/).
